# RtApi

A Ruby wrapper for the [RT ticketing system's API interface](http://requesttracker.wikia.com/wiki/REST).
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rt_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rt_api

## Usage

Just use the damn thing.
